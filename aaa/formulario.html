Text: Campo para entrada de texto simples.

Email: Campo especifico para enderecos de e-mail. O navegador valida automaticamente o formato do e-mail.

Password: Campo para senha. Os caracteres digitados são mascarados por asteriscos ou pontos.

Number: Campo para entrada de numeros.

Date: Campo para seleção de datas.

Textarea: Area de texto para entrada de varias linhas.

Radio: Botões de opcão. Apenas um pode ser selecionado por vez.
